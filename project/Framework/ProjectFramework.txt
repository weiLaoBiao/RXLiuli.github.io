W3Schools 计划:

开发原因:
	现代 Web 不断快速的发展,而人们始终缺少一个与国际上接轨的最前沿技术的学习交流之所,因此开发此网站便是为了提供给人们学习基础的 Web 相关的知识.

开发目的:
	1.制作出 W3Schools 首页,Html,CSS,JavaScript,jQuery 各一个页面并将之中文化!
	2.网站面向前端开发者初级用户.
开发流程:
	1.使用浏览器的 翻译插件 进行机翻,并将网页使用 Html+Css+js/jQuery 制作出来,然后根据原文与译文对比进行校准.

完成效果图:
	首页
	小型插件
	Html5+Css 布局

所用知识点:
    css3 弹性盒子布局
    web 字体

web 开发规范:
	1.所有的标签均为小写,并且有结束符号.
		如<img src="path"/>,<br/>,尽管你不写 / 直接写成 <img src="path"> 或者 <br> ,浏览器目前也可以正确运行,然而以后 html 可能不会再进行支持.并且遵守规范的 html 代码可以轻易转化成 xml 文档,拥有相当好的可读性.
	2.所有的 css 属性(id,class),最好采用一种命名规范并且贯彻始终.
		如中缀命名法,下划线或者驼峰命名法,目前流行的命名规范是 中缀命名法,css 官方的很多属性也都是如此.例: font-size,background-color.
	3.保持与各大浏览器的兼容性,至少要能够兼容主流浏览器.
		如(Firefox,Chrome),ie 和 safari 可以选择性兼容( Ie 是 Windows 平台独有,Safari 是 Mac 平台独有),各大国产浏览器兼容性几乎不用怎么管(基于Chrome,没有一个是独自研发内核的!).
	4.注意防止浏览器的一些基本规范不同而造成的问题.
		如 内外边距(深受其害!),强力建议清除掉不同浏览器的那些公有的属性值.
		* {
			margin: 0;
			padding: 0;
			font-size: 14px;
		}
